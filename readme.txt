A first-person player game written with 27 C# scripts in the Unity gaming engine. 

Game is playable in your browser via the Unity community gameplay website. Link to "Office Invaders" can be found here: https://play.unity.com/mg/other/my-new-microgame-191

Gameplay is simple: enter play mode by clicking Start on the home screen. Use WASD to navigate the environment, LMB to shoot a stream of bullets. Player health is limited; use walls and other obstacles to hide from enemy bullets and win the game. Player death triggers the Game Over message. 

This game was built using free assets from the Unity asset store:
